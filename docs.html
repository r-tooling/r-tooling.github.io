Overview

Welcome to r4r's documentation!

r4r is a new system for making R computational notebooks reproducible. It is designed to be an alternative to manual Docker setup and dependency management tools. Our goal with r4r is to build a reproducibility tool that is highly capable and a pleasure to use.

This documentation is split into several parts: A beginner-friendly tutorial that introduces r4r through a practical use case and a comprehensive reference that explains all of r4r's concepts and features.
