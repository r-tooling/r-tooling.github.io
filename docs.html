R4R - Overview

Welcome to R4R's documentation!

R4R is a new tool for making R computational notebooks reproducible. It is designed to be an alternative to manual Docker setup and dependency management tools. Our goal with R4R is to build a reproducibility tool that is highly capable and a pleasure to use.

This documentation is split into several parts: A beginner-friendly tutorial that introduces R4R through a practical use case and a comprehensive reference that explains all of R4R's concepts and features.
